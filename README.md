## ğŸ”§ Configuration (Optional)

Enable advanced features by adding API keys (all optional, all have free tiers):

1. Copy `.env.example` to `.env`
2. Add your API keys:
   ```
   OPENAI_API_KEY=your_key_here     # For AI conversations
   NEWS_API_KEY=your_key_here        # For news updates
   OPENWEATHER_API_KEY=your_key_here # For weather
   ```

**Get Free API Keys:**
- [OpenAI](https://platform.openai.com/api-keys) - Free trial available

- [NewsAPI](https://newsapi.org/) - Free tier: 500 requests/day
- [OpenWeatherMap](https://openweathermap.org/api) - Free tier available
